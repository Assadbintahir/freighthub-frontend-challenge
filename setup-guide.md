### For development:
- First we'll start the local server by running following command
   -    `npm run start:dev:server`
- Then start the client webpack server (which supports hot module reloading) in dev mode by following command
    -   `npm run start:dev:client`
 
 ### For prod build:
 - Run the following command. It will build the react App and start the server with the correct static folder.
    - `npm run start:prod`
